# Lesson 01: Introduction & Setup

## Definition
Kotlin is a modern, statically typed programming language developed by JetBrains. It runs on the Java Virtual Machine (JVM) and is fully interoperable with Java.

## Why it is used
It was designed to address Java's limitations: it is more concise, safer (avoids NullPointerExceptions), and fully supported by Google for Android development.

## Syntax
```kotlin
fun main() {
    println("Hello, World!")
}
```

## Real-world Analogy
Think of Kotlin as an upgraded, modern sports car compared to Java which is a reliable but older sedan. Both use the same roads (JVM), but Kotlin offers a much smoother, safer, and faster driving experience.

## Common Mistakes
* Forgetting that Kotlin files must end with `.kt`.
* Trying to run code outside the `main` function.

## Best Practices
* Always use standard Kotlin package naming rules.
* Keep the main function simple and delegate logic to helper functions.
