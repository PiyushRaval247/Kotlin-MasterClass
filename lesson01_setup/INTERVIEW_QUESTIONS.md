# Lesson 01: Interview Questions

## Beginner Questions
* **Q**: What is Kotlin?
  **A**: Kotlin is a statically-typed, modern programming language that runs on the JVM and is developed by JetBrains.

## Intermediate Questions
* **Q**: Explain Kotlin's interoperability with Java.
  **A**: Kotlin compiles down to JVM bytecode, allowing it to seamlessly call Java classes and methods, and vice-versa, without any bridge layer.
