# Lesson 06: Interview Questions

## Beginner Questions
* **Q**: Can `if` return a value in Kotlin?
  **A**: Yes, `if` is an expression in Kotlin, meaning it can return a value which can be assigned to a variable.

## Intermediate Questions
* **Q**: What are the advantages of `when` over Java's `switch` statement?
  **A**: `when` is more powerful: it doesn't need `break` statements, can check ranges, types, and can be used as an expression returning values.
