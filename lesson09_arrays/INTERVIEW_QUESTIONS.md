# Lesson 09: Interview Questions

## Beginner Questions
* **Q**: Are arrays in Kotlin fixed in size?
  **A**: Yes, arrays in Kotlin have a fixed size once initialized.

## Intermediate Questions
* **Q**: What is the difference between `Array<Int>` and `IntArray`?
  **A**: `Array<Int>` represents boxed integers (`java.lang.Integer[]`), while `IntArray` represents primitive integers (`int[]`), which is more memory efficient.
