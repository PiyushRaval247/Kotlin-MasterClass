# Lesson 17: Interview Questions

## Beginner Questions
* **Q**: What is a lambda expression?
  **A**: A lambda is a short block of code that takes parameters and returns a value, behaving like a function without a name.

## Intermediate Questions
* **Q**: What does the `it` keyword represent in a lambda?
  **A**: If a lambda has only one parameter, Kotlin allows omitting its name and references it implicitly using the keyword `it`.
