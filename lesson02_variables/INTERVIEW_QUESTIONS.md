# Lesson 02: Interview Questions

## Beginner Questions
* **Q**: What is the difference between `val` and `var`?
  **A**: `val` is read-only (immutable) once initialized, whereas `var` is mutable and can be reassigned.

## Intermediate Questions
* **Q**: Does Kotlin support type inference? Explain.
  **A**: Yes. If you initialize a variable when declaring it, Kotlin automatically detects its data type, making explicit type declarations optional.
