# Lesson 05: Interview Questions

## Beginner Questions
* **Q**: How do you read input from the console in Kotlin?
  **A**: Using `readln()` or `readLine()`.

## Intermediate Questions
* **Q**: What is the difference between `readln()` and `readLine()`?
  **A**: `readln()` returns a non-null `String` and throws an exception on EOF, while `readLine()` returns a nullable `String?` which can be null.
