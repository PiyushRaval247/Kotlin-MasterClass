# Lesson 03: Interview Questions

## Beginner Questions
* **Q**: What are the basic data types in Kotlin?
  **A**: Numbers (Byte, Short, Int, Long, Float, Double), Characters (Char), Booleans (Boolean), and Strings (String).

## Intermediate Questions
* **Q**: How does Kotlin handle type conversion compared to Java?
  **A**: In Kotlin, there is no implicit type conversion. You must explicitly convert types using functions like `.toInt()`, `.toDouble()`, etc.
