# Lesson 20: Interview Questions

## Beginner Questions
* **Q**: What is the purpose of the `finally` block?
  **A**: The `finally` block contains cleanup code that is guaranteed to run regardless of whether an exception was thrown or caught.

## Intermediate Questions
* **Q**: How are checked exceptions handled in Kotlin compared to Java?
  **A**: Unlike Java, Kotlin does not have checked exceptions. All exceptions are unchecked, meaning you are not forced by the compiler to declare or catch them.
