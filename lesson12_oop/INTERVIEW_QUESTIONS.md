# Lesson 12: Interview Questions

## Beginner Questions
* **Q**: What keyword allows a class to be inherited in Kotlin?
  **A**: The `open` keyword.

## Intermediate Questions
* **Q**: What is the difference between an Interface and an Abstract Class in Kotlin?
  **A**: A class can implement multiple interfaces but inherit from only one abstract class. Interfaces cannot store state (properties cannot have backing fields), whereas abstract classes can.
