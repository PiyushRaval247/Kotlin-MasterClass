# Lesson 11: Interview Questions

## Beginner Questions
* **Q**: What is the difference between `String` and `String?` in Kotlin?
  **A**: `String` is non-nullable (cannot hold null), while `String?` is nullable and can hold null values.

## Intermediate Questions
* **Q**: What is the Elvis operator `?:` and how does it work?
  **A**: The Elvis operator returns the left-hand expression if it is not null, otherwise it returns the right-hand expression as a default fallback.
