# Lesson 19: Interview Questions

## Beginner Questions
* **Q**: What are Generics?
  **A**: Generics are a feature that allows defining classes, functions, and interfaces with type placeholders to achieve reusability and type safety.

## Intermediate Questions
* **Q**: What does `<T : Number>` mean in Kotlin Generics?
  **A**: This is a type constraint. It specifies that the type parameter `T` must be a subclass of `Number` (like `Int`, `Double`, etc.).
