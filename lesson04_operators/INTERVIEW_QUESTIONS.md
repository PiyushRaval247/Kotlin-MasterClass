# Lesson 04: Interview Questions

## Beginner Questions
* **Q**: List the categories of operators in Kotlin.
  **A**: Arithmetic, Comparison, Logical, Assignment, Unary, and Bitwise.

## Intermediate Questions
* **Q**: What is the difference between `==` and `===` in Kotlin?
  **A**: `==` checks for structural equality (compares values using `equals()`), while `===` checks for referential equality (compares references pointing to the same object).
