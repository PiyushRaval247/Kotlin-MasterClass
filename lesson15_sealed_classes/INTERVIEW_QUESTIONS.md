# Lesson 15: Interview Questions

## Beginner Questions
* **Q**: What is a sealed class?
  **A**: A sealed class allows restricting the inheritance hierarchy to a closed set of subclasses defined in the same package.

## Intermediate Questions
* **Q**: How do sealed classes differ from enums?
  **A**: Enums can only have one instance per constant, while sealed subclasses can have multiple instances, each holding its own specific properties and internal state.
