# Lesson 18: Interview Questions

## Beginner Questions
* **Q**: What is a higher-order function?
  **A**: A function that takes another function as a parameter, returns a function, or both.

## Intermediate Questions
* **Q**: Differentiate between `apply` and `let`.
  **A**: `apply` executes the block and returns the receiver object itself, whereas `let` executes the block and returns the evaluation result of the lambda.
