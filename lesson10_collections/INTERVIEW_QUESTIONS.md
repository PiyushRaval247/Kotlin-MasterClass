# Lesson 10: Interview Questions

## Beginner Questions
* **Q**: What is the difference between a List and a Set?
  **A**: A List is ordered and allows duplicate elements, while a Set is unordered and contains only unique elements.

## Intermediate Questions
* **Q**: What is the difference between `listOf()` and `mutableListOf()`?
  **A**: `listOf()` returns an immutable/read-only list whose size and elements cannot change, while `mutableListOf()` returns a list that can be modified at runtime.
